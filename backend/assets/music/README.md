# Background Music

Place your background music files here for templates.

## Required Files

The following music files are referenced in template configurations:

- `upbeat.mp3` - Used in "highlights" template
- `ambient.mp3` - Used in "cinematic" template  
- `acoustic.mp3` - Used in "vlog" template
- `dramatic.mp3` - Used in "suspense" template

## File Requirements

- Format: MP3
- Bitrate: 128-192 kbps recommended
- Duration: 1-3 minutes (will be looped/trimmed as needed)
- License: Ensure you have rights to use the music

## Sources

You can find royalty-free music from:
- YouTube Audio Library
- Pixabay Music
- Free Music Archive
- Incompetech

## Note

If music files are missing, video generation will continue without background music.
