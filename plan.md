任务目标
请做一个可以运行的 Demo：
用户上传一段视频后，系统自动切分素材，并基于不同模板生成多条短视频结果。
最终流程：
上传视频 → 自动切分片段 → 选择 / 应用模板 → 生成短视频 → 预览 / 下载结果
核心要求
1. 支持上传视频，并在页面中预览。
2. 自动将视频切分成多个片段。
3. 提供至少 2 个不同的视频模板。
4. 系统可以基于这些模板，直接生成至少 2-3 条不同风格的短视频结果。
5. 生成结果必须是真实视频，不是文字方案或静态展示。
6. 每条结果需要支持预览，最好可以下载。
片段展示
切分完成后，每个片段简单展示即可：
- 缩略图
- 开始时间
- 结束时间
- 时长
模板示例
模板可以自行设计，例如：
- 高光混剪
- POV 文案视频
- 剧情 / 对话混剪
- 人物混剪
- 悬念引流视频
不要求模板非常复杂，但需要能体现不同剪辑逻辑，而不是同一个视频换标题。
结果展示
每条生成结果需要展示：
- 视频标题
- 使用的模板
- 视频预览
- 总时长
- 下载按钮
技术要求
技术栈语言限于：Typescript、Golong、Python。
库不限。
加分项
- 支持 9:16 竖屏输出
- 自动生成缩略图
- 自动添加字幕或大字标题
- 支持手动调整片段顺序
- 支持导出 MP4
- UI 简单清楚，非技术用户也能看懂
